# Peg Solitaire Game (CS449 - Sprint 3)

## Overview
This project is a Peg Solitaire game developed in Python using Tkinter for the GUI.  
It was built for CS449 and improved from Sprint 2 by adding new features in Sprint 3.

---

## Features

- Manual game mode (user plays the game)
- Automated game mode (game plays by itself)
- Board size options (5x5, 7x7, 9x9)
- Randomize board feature
- Game over detection
- GUI interface using Tkinter
- Unit tests for game logic

---

## Files

- `board.py` → handles board structure
- `game_logic.py` → contains game rules and logic
- `gui.py` → graphical interface (buttons, board display)
- `main.py` → runs the game
- `test_game.py` → unit tests

---

## How to Run

1. Open the project in Visual Studio
2. Run `main.py`
3. The game window will open
4. Choose:
   - Board size
   - Game mode (Manual or Automated)
5. Click "New Game" to start

---

## Game Modes

### Manual Mode
- Player clicks pieces to make moves
- Game continues until no moves left

### Automated Mode
- Game plays automatically
- Shows how the logic works

---

## Randomize Feature
- Randomizes the board during manual play
- Helps test different game states
